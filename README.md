### üìù Project Description
Wolfix is a modern e-commerce marketplace platform. This repository contains the **Frontend Client** built with Next.js, focused on high performance, SEO friendliness, and a seamless user experience.

**Note:** The server-side logic is located in a separate repository: [Wolfix.Server](https://github.com/Dmytro-Butnyk/Wolfix.Server).

### üõ† Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **State Management:** React Context API
- **Payments:** Stripe Integration
- **Styling:** CSS Modules / Global CSS
- **Icons:** Lucide React

### üöÄ Getting Started

1. **Clone the repository:**

   git clone [https://github.com/sannidoorg/wolfix.client.git](https://github.com/sannidoorg/wolfix.client.git)

3. **Install dependencies:**
```bash
npm install

```


3. **Configure Environment Variables:**
Create a `.env.local` file in the root directory and add:
```env
NEXT_PUBLIC_API_URL=your_api_url
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_key

```


4. **Run development server:**
```bash
npm run dev

```
